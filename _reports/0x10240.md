### Bybit 订单簿 API 陷阱
**2024-12-14** | 👍83 🔁8

> 发现 bybit 前端页面看到的订单本数据和后端 orderbook API 接口返回的不一致！

**核心问题**:
- `orderbook` 接口 vs `rpi_orderbook` 接口
- 前端显示用 `rpi_orderbook`
- API 下单用 `orderbook` （数据更差）

> 前端显示买一价格 3380，但 orderbook 返回买一 3370，卖出下单以 3370 成交。
>
> 这是什么奇葩逻辑？把 API 下单的用户不当人？
